Dataset is delimited as CSV values as follows;

X ; Y; Z; Pressure; GripAngle; Timestamp; Test ID

----------------
Test ID: 
0: Static Spiral Test ( Draw on the given spiral pattern)
1: Dynamic Spiral Test ( Spiral pattern will blink in a certain time, so subjects need to continue on their draw)
2: Circular Motion Test (Subjectd draw circles around the red point)
______________________________________________________________* Машинный перевод
Набор данных разделяется значениями CSV следующим образом;

X ; Y; Z; Давление; Угол захвата; Временная метка; Идентификатор теста

----------------
Идентификатор теста:
0: Статический спиральный тест (Нарисуйте по заданному спиральному рисунку)
1: Динамический спиральный тест (спиральный узор будет мигать через определенное время, поэтому испытуемым необходимо продолжить рисование)
2: Тест на круговые движения (испытуемые рисуют круги вокруг красной точки)


___________________________________________________________________________
Пояснения. 
CSV - Это формат хранения данных при котором каждый отдельный столбец отделяется запятой "," ,
а каждая новая запись (строка) отделяется переносом.

X ; Y; Z; - координаты. Давление - вероятно, степень нажатия, угол захвата - вероятно, угол под которым ведется рисунок,
Временная метка - соответвенно время в прямом смысле, идентификатор теста - один из 3 тестов.
Отдельный файл в данном случае это оцифрованные результат прохождения испытуемым набора из трех тестов  

Структура датасета. control - испытуемые без болезни паркинсона, parkinson - испытуемые с болезнью паркинсона

hw_dataset - 40 испытуемых
hw_drawings - изображения для двух тестов испытуемых
new_dataset - 43 испытуемых
